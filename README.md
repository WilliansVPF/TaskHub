# 🗂️ Projeto TaskHub

O **TaskHub** é uma aplicação **API REST** desenvolvida em **.NET 10** com o objetivo de gerenciar tarefas e projetos de forma moderna, escalável e segura.  
O foco principal é a **prática de boas arquiteturas e padrões utilizados no mercado**, indo além de simples CRUDs, explorando conceitos avançados como autenticação, validação, versionamento de API, camadas bem definidas e integração com Docker e Nginx.
